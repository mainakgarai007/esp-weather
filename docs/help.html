Help & Troubleshooting

Everything you need to connect, recover, and understand SkyCore.

Quick Start

  1. Connect to SkyCore_Setup WiFi on first boot.
  2. Open 192.168.4.1 and enter WiFi + API key + device key.
  3. Paste your device key from the onboarding screen.
  4. Visit the dashboard and pair using the ESP IP address.

Troubleshooting

  • 📡 If device shows Offline, verify WiFi credentials and power cycle.
  • �� Hold the hardware button for 3 seconds to reset WiFi.
  • 🛠️ Hold button during boot to enter Recovery Mode.
  • 🌐 Ensure OpenWeather API key is valid and not rate-limited.
  • 🧠 Check free RAM in device status if unstable.
FAQs
How do I regenerate the pairing key?

Use the Pair Device page or clear local storage to start fresh.

Why is the LED blinking red/blue?

That indicates a thunderstorm warning from the weather API.

Can I use this without internet?

The dashboard works offline for history, but live data needs internet.
