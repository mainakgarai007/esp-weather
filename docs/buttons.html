Button Guide

Hardware control shortcuts for SkyCore.

Force weather update + API refresh.

2 Press

Toggle display mode (season / weather).

5 Press

Enable Developer Mode (RAM, uptime, WiFi).

Hold

Reset WiFi credentials and reboot.
