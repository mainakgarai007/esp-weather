Update Instructions
  1. 1. Download the latest SkyCore.ino file.
  2. 2. Open Arduino IDE → select NodeMCU 1.0 (ESP-12E).
  3. 3. Install required libraries listed in the firmware header.
  4. 4. Flash via USB or OTA (WiFi update).
  5. 5. Confirm update success in the dashboard.
